# Python Inventory Management System

## 📝 Project Description
This is a comprehensive console-based Inventory Management System developed for the Python Fundamentals Assignment. The program allows users to seamlessly manage store items by adding, viewing, updating, and removing products. It is designed to handle both standard products and perishable goods (tracking expiration dates) while ensuring data is saved persistently across sessions. 

## 🚀 How to Run the Program

### Prerequisites
- Python 3.13 installed on your machine.
- No external libraries are required (the program uses the built-in `json` module).

### Execution Commands
1. Open your terminal or command prompt.
2. Navigate to the directory where the script is saved.
3. Run the following command:

```bash
python javier_coursework1.py
